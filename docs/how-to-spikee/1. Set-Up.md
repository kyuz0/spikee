# 1. Installing Spikee + Set-Up

## 1.1 Installation
1. pip
```bash
pip install spikee
```
*Note: To keep Spikee lightweight, this base installation only includes dependencies for OpenAI-compatible API endpoints (e.g., OpenAI, DeepSeek, TogetherAI, Google). If you plan to use providers for which spikee relies on extra SDKs (like AWS Bedrock, Azure, Groq, or Ollama), you must install the required extras. See [LLM Providers Document](../03_llm_providers.md) for more details:*
```bash
pip install "spikee[all]"
# Or specify individual providers, e.g., "spikee[bedrock,azure,ollama,groq]"
```

2. GitHub **(recommended)**
```bash
git clone https://github.com/ReversecLabs/spikee.git
cd spikee

python3 -m venv env
source env/bin/activate
pip install ".[all]"
```

**Development Notes**:

Using `pip install ".[all]"` will require you to reinstall spikee after making changes to the codebase. 

Alternatively, `pip install -e ".[all]"`, will create a symlink within the env, allowing you to make changes to the codebase without needing to reinstall.

## 1.2 Workspace Set-Up

```bash
mkdir workspace
cd workspace

spikee init
```
