# 1. Installing Spikee + Set-Up

## 1.1 Installation
1. pip
```bash
pip install spikee
```

2. GitHub **(recommended)**
```bash
git clone https://github.com/ReversecLabs/spikee.git
cd spikee

python3 -m venv env
source env/bin/activate
pip install .
```

**Development Notes**:

Using `pip install .` will require you to reinstall spikee after making changes to the codebase. 

Alternatively, `pip install -e .`, will create a symlink within the env, allowing you to make changes to the codebase without needing to reinstall.

## 1.2 Workspace Set-Up

```bash
mkdir workspace
cd workspace

spikee init
```
