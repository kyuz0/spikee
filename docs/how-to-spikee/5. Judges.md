# 5. Judges

A Judge is a Python script that analyses a dataset entry and the resulting target response to determine whether an attack was successful. 

Spikee includes two types of judges:
- **Basic Judges**, which use simple string matching or regex to evaluate the response.
- **LLM Judges**, which use an LLM agent.

LLM Judges are particularly useful for evaluating complex criteria, such as harmful or out-of-topic content where the response to "how to build a bomb" is not a simple string match, but may require understanding the semantics of the response.

## 5.1 Using Judges
Every dataset entry has a `judge_name` field, which determines which judge is used to evaluate that entry. For example, `seeds-simsonsun-high-quality-jailbreaks` uses `"judge_name": "llm_judge_harmful", "judge_args": ""`. 

This can be defined within seed folder creation or modified within a dataset file.

Some judges also include options, viewable using `spikee list judges`, and configurable using `--judge-options`:
```bash
# Example of using judge options to specify the LLM model used for llm_judge_harmful
spikee test --dataset .seeds-simsonsun-high-quality-jailbreaks. --target ... --judge-options "model=bedrock/qwen3-next-80b"
```

See [LLM Utilities](../../spikee/utilities/llm.py) for the standard model definition or [Built-In LLM Documentation](../03_llm_providers.md) for additional information.


## 5.2 Creating a Judge

To create a judge, extend the `Judge` or `LLMJudge` classes and implement the `judge` function.

- `Judge`: Standard judge implementation, exposing the `judge` function.
- `LLMJudge`: Extends the judge implementation, to include built-in handling of judge options and models. Use `_get_llm()` to get the LLM.

```python
from typing import List, Tuple

from spikee.templates.judge import Judge
from spikee.templates.llm_judge import LLMJudge
from spikee.utilities.enums import ModuleTag

class SampleJudge(Judge):
    def get_description(self) -> Tuple[object, List[ModuleTag], str]:
        """Returns the type and a short description of the judge."""
        return None, [], "A brief description of what this judge does."

    def get_available_option_values(self) -> Tuple[List[str], bool]:
        """Return supported attack options; Tuple[options (default is first), llm_required]"""
        return None, False

    def judge(
        self,
        llm_input,
        llm_output,
        judge_args="",
        judge_options=""
    ) -> bool:
        """
        Args:
            llm_input (str): The user prompt sent to the target.
            llm_output (str): The target's response.
            judge_args (str | list[str]): Judge specific arguments.
            judge_options (str, optional): Judge specific options.

        Returns:
            bool: True if the attack is successful, False otherwise.
        """
        # Your implementation here...

class SampleLLMJudge(LLMJudge):
    # get_available_option_values is handled by LLMJudge to select an LLM model for judging, do not redefine.
    def get_description(self) -> Tuple[object, List[ModuleTag], str]:
        """Returns the type and a short description of the judge."""
        return None, [ModuleTag.LLM], "A brief description of what this judge does."

    def judge(
        self,
        llm_input,
        llm_output,
        judge_args="",
        judge_options=""
    ) -> bool:
        """
        Evaluates whether an LLM's output is harmful.
        """
        llm = self._get_llm(judge_options or self.DEFAULT_LLM_MODEL)

        # Fail if no LLM is available
        if llm is None:
            return False
```

## 5.3 Suggested Tasks
1. Generate a `seeds-simsonsun-high-quality-jailbreaks` dataset, which uses `llm_judge_harmful`. Run a test against an LLM application, setting the judge's model.
2. (Optional) Create a custom dataset that prompts the LLM application to say a specific string, for example 'SPIKEE'. Create either a basic judge or an LLM judge that evaluates whether the target response contains the string 'SPIKEE', and test on your custom dataset.
3. Attempt to rejudge your dataset using `spikee results rejudge`.
